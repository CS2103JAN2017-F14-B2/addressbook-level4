Command Format Description
Help help Brings up user guide in another window
Helpf helpf Shows a quick summary of the command formats
Add add NAME [r/REMARKS] [d/DEADLINE] [l/LOCATION] [t/TAG]... E.g. add do 2103 project r/waiting for johnâ€™s reply d/03-03-2017 l/nus soc t/school Adds a task to KIT with specifications from the user.
List list,ls,l Lists all tasks in KIT.
Listdone listdone,ld Lists all completed tasks.
Listnotdone listnotdone,lnd Lists all ongoing tasks.
Listtag listag/ listtag/ lt/ ltag TAGNAME, lt TAGNAME E.g. lt school Lists all tasks of a certain tag.
Done done INDEX E.g. done 2 Marks the specified task as done.
Undo undo Undo the previous action.
Edit edit NAME [r/REMARKS] [d/DEADLINE] [l/LOCATION] [t/TAG]... E.g. edit 1 r/finish up user guide l/home Edits an existing task.
Find find KEYWORD E.g. find home, find bread and butter Finds a particular task.
Findexact finde/ fexact/ fe/ findexact KEYWORD(s). E.g. findexact do homework, fe change USD Finds a particular task with more detailed specification.
Delete delete INDEX E.g. delete 3 Deletes the specified task.
Clear clear Clears all entries.
Exit exit Exits the program.
Save save PATHNAME E.g. save C:\Program Files Saves the file to a specified location.
