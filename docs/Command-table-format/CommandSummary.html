Command Format Alternate Formats Description
Help help h Brings up user guide in another window
Help Format helpformat helpf, summary, hf Shows a quick summary of the command formats
Add add NAME [r/REMARKS] [s/STARTDATE] [s/ENDDATE] [l/LOCATION] [t/TAG]... - Adds a task to KIT with specifications from the user.
List list ls, l Lists all tasks in KIT.
List Done listdone ld Lists all completed tasks.
List Not Done listnotdone listundone, lnd Lists all ongoing tasks.
List Tag listtag TAGNAME listag TAGNAME, ltag TAGNAME, lt TAGNAME Lists all tasks of a certain tag.
Done done INDEX d INDEX Marks the specified task as done.
Undone undone INDEX notdone INDEX, ud INDEX, nd INDEX Marks the specified task as not done.
Undo undo uhoh Undo the previous command.
Edit edit NAME [r/REMARKS] [s/STARTDATE] [e/ENDDATE] [l/LOCATION] [t/TAG]... Edits an existing task.
Find find KEYWORD... f KEYWORD... Finds tasks that contains at least one keyword.
Find Exact findexact KEYWORD... fexact KEYWORD..., finde KEYWORD..., fe KEYWORD... Finds tasks that contains all keywords.
Delete delete INDEX Deletes the specified task.
Clear clear Clears all tasks.
Exit exit Exits the program.
Save save PATHNAME Saves the file to a specified location.
